## Requirements
You need to have installed to use this:
- Node.Js
- NPM
## Use it:
### First start
Before you start it, run `npm i`
### Start
Run `node app.js`

## BUILD IT 
run these commands:
- `npm i -g gulp-cli` on linux `sudo npm -g gulp-cli`
- `npm i -D`
- `npm run build`
